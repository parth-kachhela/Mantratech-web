# 💼 MantraTechSystem - Full Stack IT Company Website

A professional, full-stack business website built for **MantraTechSystem**, showcasing IT services, company capabilities, leadership team, and an integrated admin dashboard for contact management.

---

## 🌐 Live Preview

> https://mantratechsystem.netlify.app  
> Backend hosted on: https://mantratech-web.onrender.com

---

## 🧰 Tech Stack

### Frontend:
- ✅ React JS (TypeScript)
- ✅ Tailwind CSS
- ✅ Framer Motion (for animations)
- ✅ React Router

### Backend:
- ✅ Node.js + Express (TypeScript)
- ✅ PostgreSQL (via Prisma ORM)
- ✅ CORS & RESTful API

---

## 📂 Key Features

- 🏠 Elegant Home Page with Hero, Services, Vision
- 💼 About Us Page with video banner, leadership & animated team section
- ⚙️ Services Page showing all offerings in stylish card layout
- 📞 Contact Us Form with validations + embedded map
- 👨‍💻 Admin Panel with:
  - Sidebar & Navbar
  - Customer contact data list
  - Delete contact functionality
- 🎯 Careers Page styled like a product-level page
- 🌍 SEO-optimized for local targeting (Veraval, Junagadh, Rajkot)
- 📱 Fully Responsive (Mobile + Tablet + Desktop)
